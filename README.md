# 获取微信公众号图文页面中的视频URL

## 灵感

设计灵感来自于我的高中化学老师，也是我的班主任。有一天她想提取我们学校的公众号发的一个视频，但一直不知道如何提取。我通过查看网页源代码帮她找出了视频URL，然后把视频下载到了本地。自那以后我一直在想，我为什么不能设计一个程序，来帮助老师们获取微信公众号发出的视频的URL呢？

于是我就设计了这样的一个程序，在此我得感谢我的高中班主任。

## 使用方法

* Download ZIP，然后解压上传到你的服务器

* 直接访问页面即可
